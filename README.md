# Juicy Club Naturals

A modern, fast, and production-ready local delivery platform for targeted natural problem-solving shots and revitalizing fruit juices. Focus on Beauty (Glass Skin), Gut Health, and Anti-Aging.

## Features
- Supabase Authentication
- Razorpay Payment Gateway
- Mobile-first responsive design
- Shopping Cart and Admin Dashboard

## Local Development
1. Create a `.env` file with required variables.
2. Run `pip install -r requirements.txt`
3. Run `python app.py`

## Deployment
This project is configured for deployment on Render.com or Heroku.
Use the start command `gunicorn app:app`.
