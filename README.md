# Juicy Club Naturals

A modern, fast, and production-ready e-commerce platform for premium functional wellness shots and targeted results juices. Focus on Radiance, Vitality, and Ageless Beauty.

## Features
- Supabase Authentication
- Razorpay Payment Gateway
- Mobile-first responsive design
- Shopping Cart and Admin Dashboard

## Local Development
1. Create a `.env` file with required variables.
2. Run `pip install -r requirements.txt`
3. Run `python app.py`

## Deployment
This project is configured for deployment on Render.com or Heroku.
Use the start command `gunicorn app:app`.
